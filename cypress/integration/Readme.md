

## How to run Install  [Cypress. ](https://docs.cypress.io/guides/getting-started/installing-cypress.html#System-requirements)
```
* cd /your/project/path
* npm install cypress --save-dev
```


## Open Cypress:
```
* ./node_modules/.bin/cypress open
```


## Run cypress:
```
* node_modules/.bin/cypress open
```
